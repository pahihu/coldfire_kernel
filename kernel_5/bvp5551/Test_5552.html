Interface version and function

Read the function and version number from backplane dual port memory from S2. See Test555_from_bcm file for the equivalent function from the BCM. 

 
: read_version ( -- version )
	[ PORT P._interface_version + ] LITERAL W@ ;

: read_id ( -- function_id )
	[ PORT P._function_id + ] LITERAL W@ ;


: .id ( -- )
	read_version cr ." interface_version - " .
	read_id cr ." function_id - " . send
;
