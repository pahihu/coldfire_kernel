RFC1014 XDR
RFC1057 RPC
RFC1094 NFS

Thre spec looks hard doesn't it. RFC1014 describes a method to describe data, nothing more. RFC1057 describes a data structure to send for a RPC and the reply to expect. RFC1094 describes the functions available for NFS.

In the end it is all pretty simple, send a UDP datagram to port 111, with data as speced in: RFC1057 RPC and you will get back the required data, unpack the data and put it on the stack and it is done.
