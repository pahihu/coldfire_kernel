4dup

Duplicate the top four stack items. 

 
	: 4dup  ( a b c d -- a b c d )
		2OVER 2OVER 
	;
jump

Third stack item copied to the top. 

 
	.S .( jump)
	: jump ( a b c -- a b c a )  
		two PICK 
	;
char!

Same as C! available because C! is often used for byte operations. 

 
	: char! ( character addr -- )
		C!
	;  inline \ renamed zero over head
char@

Same as C@ available because C@ is often used for byte operations. 

 
	: char@ ( addr -- character )
		C@
	; inline
byte!

ANSI forth has no byte words. 

 
	: byte! ( byte addr -- )
		C!
	;  inline \ renamed zero over head
byte@

ANSI forth has no byte words. 

 
	: byte@ ( addr -- byte )
		C@
	; inline

You use CHAR to get from characters to bytes, you use byte>char get from bytes to char 

 
	: bytes>chars ( n -- n2)
	; inline
