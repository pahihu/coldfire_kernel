The base is used for number to conversion.

ANS 6.1.1170 DECIMAL ( --)

CORE

( -- )

Set the numeric conversion radix to ten (decimal). 

 
	: DECIMAL ( --)   
		$0A BASE ! 
	;
ANS 6.2.1660 HEX ( --)

( -- )

CORE EXT

Set contents of BASE to sixteen. 

 
	: HEX ( --)
		$10 BASE !
	;
binary (--)

Set the contents of BASE to two. 

 
	: binary  ( --)   
		#02 BASE ! 
	;
octal (--)

Set the contents of BASE to eight. 

 
	: octal   ( --)   
		#08 BASE ! 
	;
